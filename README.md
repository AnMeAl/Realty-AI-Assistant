# Сервис автоматической проверки объявлений о недвижимости

### Описание проекта
Сервис для автоматической проверки объявлений о продаже/аренде квартир. Система получает на вход ссылку на объявление или скриншот параметров, проводит "расследование" и выдает сводный отчет о справедливости цены.

### Цель проекта
Разработать систему, которая поможет пользователям понять, насколько цена в объявлении соответствует рыночной, выявить потенциальные проблемы и скрытые особенности квартиры.

### Как это работает
1. **Ввод данных**: пользователь отправляет ссылку на объявление или скриншот
2. **Парсинг данных**: система извлекает параметры квартиры (площадь, этаж, метро, цена и т.д.)
3. **Анализ**: сравнение с рыночными данными, выявление аномалий
4. **Отчет**: формирование сводного отчета с рекомендациями

### Текущая реализация
На данный момент реализован модуль парсинга данных с Яндекс.Недвижимости:
- Сбор информации об однокомнатных квартирах в Москве
- Извлечение ключевых параметров: площадь, цена, этаж, метро, адрес
- Структурирование данных для дальнейшего анализа

### Технологический стек
- **Python 3.11ю.+** - основной язык разработки
- **uv** - пакетный менеджер
- **requests / BeautifulSoup4** - парсинг HTML
- **DVC** - управление данными
